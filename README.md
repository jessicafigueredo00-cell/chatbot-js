# 🤖 Chatbot de Atendimento (JavaScript)

## 📖 Descrição

Simulação de um sistema de atendimento automatizado baseado em regras, desenvolvido em JavaScript. O projeto representa um fluxo simples de interação com o usuário, semelhante a um chatbot utilizado em canais como WhatsApp.

---

## 🎯 Objetivo

Criar um fluxo de atendimento automatizado capaz de responder às solicitações do usuário com base em opções pré-definidas.

---

## ⚙️ Funcionalidades

* Menu de opções de atendimento
* Respostas automáticas baseadas na escolha do usuário
* Tratamento de entradas inválidas
* Simulação de fluxo de atendimento

---

## 🛠️ Tecnologias Utilizadas

* JavaScript

---

## 🧠 Lógica do Sistema

O sistema recebe uma entrada do usuário e, por meio de estruturas condicionais, retorna uma resposta correspondente à opção escolhida.

---

## 💻 Exemplo de Uso

O usuário escolhe uma opção:

1 - Horário
2 - Preço
3 - Falar com atendente

O sistema retorna a resposta automaticamente com base na escolha.

---

## 🚀 Resultados

O projeto demonstra a aplicação de lógica de programação para construção de fluxos de atendimento automatizados, simulando o comportamento de chatbots utilizados em ambientes reais.

---

## 💡 Observações

Projeto desenvolvido com foco em lógica de automação e simulação de atendimento.
